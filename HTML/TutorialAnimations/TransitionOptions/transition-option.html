Transitions options

Transition property and option that can be used with this property

  • transition-timing-function: deals with how the change in property will occur

    For example will the background color change fast at the beging and slow at the end, or it will be smooth all the way. The transition timing function have 5 different values that can be used.

    • ease: the change starts slow and then go faster then finishes slow (default)
    • linear: the change occurs at the same rate
    • ease-in: the changes starts slow and then go faster before the end
    • ease-out: the change starts fast then slows down before the end
    • ease-in-out: the change starts slow then go faster then finishes slow (kind like ease, there isn't a big difference between them)

  • translation-delay

    • The time before the transition starts
    • Accepts values in seconds and milliseconds
